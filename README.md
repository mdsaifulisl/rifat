# Personal Portfolio Website

This project is a personal portfolio site built to showcase my skills, work experience, and contact information. It includes a hero area, about section, services, skills, portfolio work, and a contact form.

## Features

- Responsive design using Bootstrap
- Typing animation in the hero section
- Modern UI with smooth layout
- Service overview
- Skills section with icons
- Work experience timeline
- Contact form with JavaScript handling
- Footer with social links

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome
- Google Fonts

## Sections Included

### 1. Header
Navigation menu with a logo and responsive mobile toggle.

### 2. Hero
Intro area with a typing effect that displays your name dynamically.

### 3. About
A short description of who I am and what I do.

### 4. Services
List of services I provide as a web developer.

### 5. Skills
My core skills including:
HTML, CSS, JavaScript, Bootstrap, React, Node.js, Express.js, MongoDB, MySQL and more.

### 6. Work Experience
Timeline and list of previous work roles or freelance projects.

### 7. Contact
Contact form with JavaScript validation and submission handling.

### 8. Footer
Social links and copyright text.

## How to Use

1. Clone or download the project files.
2. Open `index.html` in your browser.
3. Update images, text content, and links as needed.
4. Customize style files inside the `style` folder.
5. Modify JavaScript logic inside `script/script.js`.
